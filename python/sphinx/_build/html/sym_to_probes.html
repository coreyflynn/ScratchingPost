sym_to_probes¶

script to convert a list of gene symbols to a list of probes associated with those genes

STAND ALONE USAGE: sym2probes symbol_list_path probe_list_path

REQUIRED ARGUMENT DEFINITIONS: symbol_list_path: the file containing the list of symbols to convert probe_list_path: the file ot be written containing the probes list
