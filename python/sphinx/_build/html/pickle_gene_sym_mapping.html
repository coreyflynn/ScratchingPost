pickle_gene_sym_mapping¶

function to create a pickled dictionary of gene symbol keys and probset values from a text file with probe/symbol pairs created with ProbesetGeneSymbolPairsFromChip.py

USAGE: python pickleGeneSymMapping inputFilePath outputFilePath

VARIABLE DEFINITIONS: inputFilePath: the path to the .txt file to be operated on outputFilePath: the path to the file to be written to
