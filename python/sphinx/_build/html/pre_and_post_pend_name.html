pre_and_post_pend_name¶

utility function to add a constant string to the both the beginning and the end of each string on the lines of the input text file

USAGE: python preAndPostPendName inputFilePath preString postString outputFilePath

VARIABLE DEFINITIONS: inputFilePath: the path to the file of strings (one per line) to be operated on preString: the string to add to the begining of each line in the input file postString: the string to add to the end of each line in the input file outputFilePath: the path to the file to be written as output
