sqlite_utils¶

methods for the creation and manipulation of sqlite databases using sqlite3 Created on Mar 7, 2012 @author: Corey Flynn, Broad Institute

cjf.cmap.util.sqlite_utils.add_row_to_table(db_path, table_name, row_data)[source]¶

adds the list of data given by row_data to table_name in db_path. The data in row_data is assumed to be of the right format for the given table

cjf.cmap.util.sqlite_utils.add_table(db_path, table_name, table_list)[source]¶

create a new table in the database at db_path. The name of the table is given in table_name and the fields are specified by the entries in table_list. Table_list entries must be of the form (‘name’,’sql_data_type’)

cjf.cmap.util.sqlite_utils.create_db(db_path)[source]¶

create an empty sqlite database at the location specified in db_path.

cjf.cmap.util.sqlite_utils.ensure_sqlite(db_path)[source]¶

ensures that the given db_path ends in .sqlite in order to standardize database naming

cjf.cmap.util.sqlite_utils.run_query(db_path, query)¶

returns a list of all of the objects returned from the given query into the database
