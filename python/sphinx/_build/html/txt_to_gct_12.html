txt_to_gct_12¶

txt2gct12 converts a data file in txt format to one in gct version 1.2 format. file formats can be viewed at (http://www.broadinstitute.org/cancer/software/genepattern/tutorial/gp_fileformats.html)

USAGE: txt2gct12 inputFile outputFile

INPUT VARIABLES: inputFile: the .txt file to be converted outputFile: the .gct file tobe written

NOTES: this script is a command line wrapper around gctUtils.txt2gct12

Author: Corey Flynn, Broad institute, 2011
