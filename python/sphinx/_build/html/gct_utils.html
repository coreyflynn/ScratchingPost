gct_utils¶

Utility functions related to handling gct files

cjf.cmap.util.gct_utils.gct_13_to_12(version13File, version12File)[source]¶

gct_13_to_12 reverts a version 1.3 .gct file to a 1.2 .gct file compatible for use with utilities that are not updated for use with version 1.3 files.

USAGE: gct_13_to_12(version13File,version12FileLocation)

INPUT VARIABLES: version13File: the version 1.3 .gct file to convert version12File: the path to which the version 1.2 .gct file will be written

Author: Corey Flynn, Broad institute, 2011

cjf.cmap.util.gct_utils.txt_to_gct_12(inputFilePath, outputFilePath)[source]¶

txt_to_gct_12 converts a data file in txt format to one in gct version 1.2 format. file formats can be viewed at (http://www.broadinstitute.org/cancer/software/genepattern/tutorial/gp_fileformats.html)

USAGE: txt_to_gct_12 inputFile outputFile

INPUT VARIABLES: inputFile: the .txt file to be converted outputFile: the .gct file tobe written

NOTES: this script is a command line wrapper around gctUtils.txt_to_gct_12

Author: Corey Flynn, Broad institute, 2011

cjf.cmap.util.gct_utils.update_to_parse_gct2(inputDirectory, outputDirectory)[source]¶

Utility function to update matlab files containing references to parse_gct and its produced fields to files that use parse_gct2 and its produced fields.

USAGE: update_to_parse_gct2(inputDirectory,outputDirectory)

VARIABLE DEFINITIONS: inputFilePath: the path to the directory of .m files to be operated on outputFilePath: the path to directory to be written to
